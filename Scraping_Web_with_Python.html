Web Scraping pada Website Bukalapak Menggunakan Python¶

Web Scraping adalah mengambil sebuah dokumen semi-terstruktur dari internet, umumnya berupa halaman-halaman web dalam bahasa markup seperti HTML atau XHTML, dan menganalisis dokumen tersebut untuk diambil data tertentu dari halaman tersebut. Web scraping banyak digunakan untuk mengambil data cuaca yang nantinya akan digunakan untuk prakiraan cuaca, kemudian data film dari sebuah website yang kemudian digunakan untuk analisis bisnis sebuah rumah produksi film, ada pula yang mengambil data harga sebuah produk dari marketplace yang nantinya digunakan untuk membandingkan harga produk tersebut.

Bukalapak merupakan sebuah e-commerce jual beli online yang menyediakan berbagai macam barang untuk kebutuhan sehari hari via website maupun aplikasi mobile. Web scraping pada website Bukalapak kali ini akan mengambil beberapa data dari produk populer yang ada pada kategori Kecantikan. Data yang berhasil diambil kemudian akan dibuat visualisasi datanya untuk dianalisis lebih lanjut.

Sebelum memulai scraping web menggunakan python ada beberapa hal yang perlu disiapkan. Pertama, kita harus menginstall library yang diperlukan untuk scraping web yaitu "request" dan "bs4". Pada library "bs4" kita akan menggunakan sebuah fungsi yang bernama BeautifulSoup yang berguna untuk proses pengambilan data pada website. Untuk menginstall package tersebut, lakukan langkah di bawah ini pada Anaconda prompt di PC anda:

  • pip install requests
  • pip install bs4

csv=open("db_bukalapak.csv", 'w')

#membuat header (nama kolom) pada file db_bukalapak.csv
headers = "nama_produk,harga,penjual,kota,rating_produk,jml_review\n"
csv.write(headers)

#scraping 3 halaman yang ada di website bukalapak 
for page in range(1,8):
    #link yang akan di scrape
    base_url = 'https://www.bukalapak.com/products/s?page='+str(page)+'&search%5Bbrand_seller%5D=0&search%5Bcity%5D=&search%5Bcourier%5D=&search%5Bfree_shipping_coverage%5D=&search%5Bhashtag%5D=&search%5Binstallment%5D=0&search%5Bkeywords%5D=lipstik&search%5Bnew%5D=1&search%5Bprice_max%5D=&search%5Bprice_min%5D=&search%5Bprovince%5D=&search%5Brating_gte%5D=1&search%5Brating_lte%5D=5&search%5Btodays_deal%5D=0&search%5Btop_seller%5D=0&search%5Bused%5D=1&search%5Bwholesale%5D=0&utf8=%E2%9C%93'
    #print(base_url)

    #Request URL dan parser web 
    uClient = uReq(base_url)
    page_html = uClient.read()
    uClient.close()
    page_soup = BeautifulSoup(page_html, "html.parser")
    all_product = page_soup.find_all("div", class_ = "product-card")
    #print(len(all_product)) 
    
    #list data apa saja yang akan diambil dari produk 
    for item in all_product:
        
        # nama  dan link product
        product_name = item.find("a", {"class":"product__name line-clamp--2 js-tracker-product-link qa-list"})
        #product_link = 'https://www.bukalapak.com' + str(product_name.get('href'))
        product_name = product_name.text.replace('\n', "").strip()        

        # harga
        product_price = item.find("span", {"class":"amount positive"})
        product_price = product_price.text.replace('\n', "").strip()
        product_price = 'Rp ' + product_price        
        
        #penjual        
        seller_name = item.find("h5", {"class":"user__name"})
        seller_name = seller_name.text.replace('\n', "").strip()
        
        #kota pengiriman
        seller_city = item.find("span", {"class":"user-city__txt"})
        seller_city = seller_city.text.replace('\n', "").strip()    
        
        #rating 
        rating = item.find("div", {"class":"product__rating"})
        rating = rating.text.replace('\n', "").strip()
        rating = rating[0:3]
        
        #jumlah review
        review = item.find("div", {"class":"product__rating"})
        review = review.text.replace('\n', "").strip()
        review = review[3:]
        
        #write data yang sudah di scrape
        csv.write(product_name+','+ product_price+','+ seller_name+','+ seller_city+','+ rating+','+ review+'\n')
        #csv.write()
csv.close()

Kesimpulan¶














Scrapping merupakan salah satu cara untuk mengoleksi/mengumpulkan data ketika kita tidak punya akses langsung untuk mengambil data dari sebuah situs. Metode ini cukup efektif dalam mengumpulkan data karena cukup mudah dan tidak membutuhkan biaya banyak (bahkan bisa tanpa biaya). Akan tetapi tidak semua situs web dapat di scrap karena beberapa situs bersifat tertutup. Bukalapak menjadi salah satu website yang dapat diambil beberapa datanya sehingga kita dapat melakukan analisis terhadap sampel data yang kita ambil. Selain itu, data hasil scrapping juga dapat disimpan ke dalam berbagai format yang semakin memudahkan kita untuk melakukan analisis. Data yang berhasil diambil dari website Bukalapak pada latihan kali ini berjumlah 350 data yang memuat nama produk, penjual, kota pengiriman, harga, rating produk, dan jumlah review produk.
